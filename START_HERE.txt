QUICK START (5 steps):

1. INSTALL DEPENDENCIES
   Run: pip install -r requirements.txt

2. VERIFY INSTALLATION

3. RUN TESTS
   Run: python tests/test_strategies.py

4. SETUP IBKR TWS

5. START PAPER TRADING
   Run: python main.py
